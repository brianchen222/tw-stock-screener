# 台股做多形態選股器

以**趨勢線 + 形態學**自動篩出「適合做多、且一張投資金額 2 萬 ~ 10 萬」的台股標的,
並把每檔的 K 線圖(含趨勢線、頸線、形態關鍵點標註)輸出成 PNG。

> ⚠️ **務必先讀「重要限制」一節**。程式做的是「候選標的偵測 + 信心分數」,
> **不是**保證準確的買賣訊號;最終一定要自己看 K 線圖人工確認。

---

## 偵測內容

### 做多形態(形態學)
| 形態 | 說明 | 可靠度 |
|------|------|--------|
| 雙重底 / W 底 | 兩個等高底 + 突破中間頸線 | ★★★ 最可靠 |
| 頭肩底 | 左肩-頭(最低)-右肩 + 突破頸線 | ★★☆ |
| 複合頭肩底 | 多重肩的頭肩底 | ★☆☆ 主觀,須人工複核 |
| 跳空向上 / 雙跳空 | 向上跳空缺口(動能輔助) | 輔助訊號 |

> 註:你提到的「頭肩頂、雙重頂 M 頭」屬於**做空**形態,依你的需求(只做多)已排除,
> 不會出現在結果中。

### 趨勢線
- **上升支撐**(低點連線向上,順勢做多)
- **下降阻力突破**(由空翻多)
- **橫向支撐 / 壓力突破**(箱型整理突破)
- **上升雙軌道**(平行通道)

### 篩選條件
- **價格**:NT$20 ~ NT$100(= 一張 2 萬 ~ 10 萬,台股一張 1000 股)
- **流動性**:近 20 日均量 ≥ 500 張(剔除冷門股)
- 命中 ≥ 1 個做多形態或趨勢訊號才納入,並依綜合分數排序

### 進場 / 停損 / 資金管理(風險控管)
每檔命中標的會自動計算進場、停損與部位口數:
- **進場價 / 停損價**(先有停損才有進場):
  - 有向上跳空缺口且仍是支撐(缺口在現價之下)→ 進場 = 缺口上緣、停損 = 缺口下緣
  - 否則用形態 → 進場 = 頸線突破價、停損 = 形態支撐(W 底第二底 / 頭肩底右肩)
- **第一批口數**(預設總資金 30 萬、風險 1%):
  - 風險金額 = 投資資金 × 1%
  - 每張停損金額 = (進場 − 停損) × 1000
  - 建議口數 = min(風險金額 ÷ 每張停損金額, 資金 ÷ 每張成本)
  - 實際風險比例 = 口數 × 每張停損金額 ÷ 投資資金
  - 若 1 張就超過 1% 風險 → 標記「停損過寬」、建議口數 0(需自行縮小停損或接受較高風險)
- **加碼(金字塔)**:自動建議「+2R 加碼、停損上移到 +1R」情境,並依你的公式算可加碼口數;
  第一批鎖住的獲利可覆蓋加碼風險時標記為「無風險加碼」。
- 參數都在 [`config.py`](config.py):`TOTAL_CAPITAL`、`RISK_PCT`、`ADD_AT_R`、`RAISE_STOP_TO_R`。

### 加碼計算器(手動,現場用)
加碼進場價/停損價是價格走高後現場決定的,可用獨立計算器:
```bash
# 只算第一批 + 自動建議加碼
python calc.py --entry 35.45 --stop 33.20

# 自訂加碼價/停損(現場決定時)
python calc.py --entry 35.45 --stop 33.20 --add-entry 39.0 --add-stop 37.5

# 自訂總資金 / 風險%
python calc.py --entry 35.45 --stop 33.20 --capital 500000 --risk 0.01
```

## 安裝

```bash
cd stock-screener
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## 執行

```bash
source .venv/bin/activate

python run.py                     # 全上市+上櫃掃描(完整,約 5~10 分鐘)
python run.py --market listed     # 只掃上市
python run.py --limit 100         # 只掃前 100 檔(快速測試)
python run.py --max-charts 20     # 最多畫 20 張圖
python run.py --no-charts         # 只出 CSV 不畫圖
python run.py --min-price 30 --max-price 80   # 自訂價格區間
```

> 第一次跑要連網下載資料(較久);資料會快取在 `data/`,12 小時內再跑會很快。

## 輸出
- `output/report.html` — **互動式網頁報告**(可搜尋/篩選/排序的標的卡片,附 K 線圖、進場/停損/口數/加碼)。用瀏覽器開啟:
  ```bash
  open output/report.html        # macOS
  ```
- `output/screener_results.csv` — 命中清單(代號、名稱、收盤、一張成本、分數、形態、進場/停損/口數/風險、趨勢、備註)
- `output/charts/*.png` — 分數最高的前 N 檔 K 線圖(含趨勢線、形態標記、進場/停損標籤)

### 網頁報告功能
- 上方統計:命中數、可進場數(口數≥1)、已突破數
- 篩選:搜尋代號/名稱、市場別、形態別、「只看可進場」「只看已突破」「只看有圖」
- 排序:依分數 / 風險% / 一張成本 / 建議口數
- 每張卡片:K 線圖(點擊放大)、形態標籤(✓突破/形成中)、進場/停損/口數/投入/風險%/加碼
- 停損過寬者會標示警告(1 張即超過設定風險)

## 重要限制(請務必理解)

1. **形態偵測是演算法近似,不等於人眼判讀。** 同一段 K 線,程式畫的頸線/趨勢線可能與你
   手畫的不同;「複合頭肩底」尤其主觀,結果僅供縮小範圍,**請看圖人工複核**。
2. **這不是買賣建議。** 分數高 ≠ 會漲。形態學是機率性工具,需搭配基本面、量能、大盤環境。
3. **突破可能失敗(假突破)。** 程式標「✓突破」只代表收盤站上頸線,不保證續強。
4. **資料來源為免費 API(yfinance / FinMind),** 偶有缺漏或延遲;除權息未還原可能影響長期型態。
5. **僅供研究與學習,風險自負。**

## 參數調整
所有門檻集中在 [`config.py`](config.py):價格區間、均量、回看天數、各形態容許值、
轉折靈敏度(`PIVOT_WINDOW`,調大=只看大波段、調小=更敏感)等,可依需求微調。

## 程式結構
```
run.py            CLI 入口(選股)
calc.py           CLI 入口(手動加碼計算器)
config.py         所有參數
src/universe.py   台股清單(twstock)
src/fetch.py      OHLCV 下載 + 快取(yfinance 主、FinMind 備援)
src/pivots.py     轉折點偵測(所有形態的基礎)
src/trendlines.py 趨勢線
src/patterns.py   形態學(W底/頭肩底/複合頭肩底/跳空)
src/risk.py       進場/停損/口數/加碼(資金管理)
src/charting.py   K 線圖繪製 + 標註(含進場/停損標籤)
src/report.py     互動式 HTML 網頁報告產生器
src/screener.py   主流程 + 評分 + 資金管理 + 報告
```
